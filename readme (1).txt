WHAT IS THIS?
-------------
Chemistry Practice 3.0 is a fully offline, browser-based
chemistry learning platform. No internet connection required
after the first time you open the file. All data is stored
locally in your browser (localStorage).

FILES INCLUDED
--------------
  index.html       — Main application (open this!)
  how_to_use.html  — Step-by-step usage guide
  readme.txt       — This file

HOW TO OPEN
-----------
Simply double-click index.html to open it in your browser.
Recommended browsers: Chrome, Edge, Firefox, Safari (latest).
